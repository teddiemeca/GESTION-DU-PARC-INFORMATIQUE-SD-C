# Gestion du Parc Informatique - SD-C

Ce projet est une application console développée en **langage C**. Elle a été conçue dans le cadre d'un projet académique pour permettre la gestion simplifiée d'un inventaire d'ordinateurs.

## Description
L'application permet de manipuler une base de données d'ordinateurs avec les fonctionnalités suivantes :
    Ajouter un ordinateur     
    Lister les ordinateurs      
    Afficher les details        
    Supprimer un ordinateur     
    Repartir par specialite     
    Repartir par marque         
    Repartir par RAM            
    Classer par vitesse CPU     
    Calculer la valeur totale

## Structure du Projet
Le projet est organisé en trois fichiers principaux :
* `ordinateur.h` : Définitions des structures et prototypes des fonctions.
* `ordinateur.c` : Implémentation des fonctions de gestion.
* `main_ordinateur` : Point d'entrée du programme
